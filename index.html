راهنمای اتصال در Android

آموزش نصب و راه‌اندازی V2ray روی دستگاه‌های اندروید

گام 1️⃣: نصب اپلیکیشن

🔹 اپلیکیشن پیشنهادی: V2rayNG

📥 دانلود:

از لینک زیر آخرین نسخه را دانلود کنید:

دانلود V2rayNG از GitHub

یا از Google Play:

نصب از Play Store

گام 2️⃣: افزودن اشتراک

روش 1:

  1. اپلیکیشن V2rayNG را باز کنید
  2. روی آیکون ➕ (پلاس) بزنید
  3. گزینه "Import Config from Clipboard" را انتخاب کنید
  4. لینک اشتراک خود را کپی کرده و وارد کنید

روش 2:

  1. روی آیکون ☰ (منوی همبرگری) بزنید
  2. گزینه "Subscription Setting" را انتخاب کنید
  3. روی ➕ بزنید و لینک اشتراک را وارد کنید
  4. روی "Update Subscription" بزنید

گام 3️⃣: اتصال

  1. یکی از سرورها را انتخاب کنید
  2. روی دکمه اتصال (گوشه پایین راست) بزنید
  3. اجازه VPN را بدهید

✅ تمام! حالا به اینترنت آزاد متصل هستید!!

نکات مهم:
  • برای بهترین سرعت، سرورهای مختلف را تست کنید
  • گزینه "Auto Select" را برای انتخاب خودکار بهترین سرور فعال کنید
  • برای بروزرسانی سرورها، "Update Subscription" را بزنید
  • در صورت قطعی اتصال، سرور را تغییر دهید

راهنمای اتصال در iOS/iPadOS

آموزش نصب و راه‌اندازی V2ray روی دستگاه‌های اپل

گام 1️⃣: نصب اپلیکیشن

🔹 اپلیکیشن پیشنهادی: Streisand

📥 دانلود:

دانلود Streisand از App Store

یا گزینه‌های دیگر:

  • V2Box
  • Shadowrocket (پولی - $2.99)

گام 2️⃣: افزودن اشتراک

روش 1 (پیشنهادی):

  1. لینک اشتراک خود را کپی کنید
  2. اپلیکیشن Streisand را باز کنید
  3. روی ➕ یا "Add Subscription" بزنید
  4. لینک به صورت خودکار شناسایی می‌شود
  5. روی "Add" یا "تایید" بزنید

روش 2:

  1. در اپلیکیشن، به تنظیمات بروید
  2. "Subscription" → "Add"
  3. لینک اشتراک را وارد کنید

گام 3️⃣: اتصال

  1. یکی از سرورها را انتخاب کنید
  2. روی دکمه اتصال بزنید
  3. اجازه VPN را بدهید (اولین بار)
  4. رمز عبور یا Face ID/Touch ID را وارد کنید

✅ تمام! حالا به اینترنت آزاد متصل هستید!

نکات مهم:
  • برای بروزرسانی سرورها، روی دکمه Refresh بزنید
  • در صورت قطعی، سرور دیگری را امتحان کنید
  • حتماً اپدیت‌های اپلیکیشن را نصب کنید
  • از آخرین نسخه iOS استفاده کنید

راهنمای اتصال در Windows

آموزش نصب و راه‌اندازی V2ray روی سیستم‌های ویندوز

گام 1️⃣: نصب اپلیکیشن

🔹 اپلیکیشن پیشنهادی: Hiddify Next

📥 دانلود:

دانلود Hiddify-Next

(فایل Hiddify-Windows-Setup-x64.exe را دانلود کنید)

یا گزینه دیگر:

  • V2rayN (فایل v2rayN-With-Core.zip)

گام 2️⃣: نصب

  1. فایل دانلود شده را اجرا کنید
  2. مراحل نصب را طی کنید
  3. اپلیکیشن را باز کنید

گام 3️⃣: افزودن اشتراک

در Hiddify Next:

  1. روی "➕ New Profile" کلیک کنید
  2. گزینه "Add from Clipboard" را انتخاب کنید
  3. لینک اشتراک را کپی کنید
  4. روی "Add" کلیک کنید

در V2rayN:

  1. روی "Subscription" → "Subscription Setting" کلیک کنید
  2. روی "Add" کلیک کنید
  3. لینک اشتراک را در فیلد URL وارد کنید
  4. روی "Confirm" کلیک کنید
  5. روی "Update Subscription" کلیک کنید

گام 4️⃣: اتصال

  1. یکی از سرورها را از لیست انتخاب کنید
  2. روی دکمه Connect یا Enter کلیک کنید
  3. آیکون در System Tray سبز می‌شود

✅ تمام! حالا به اینترنت آزاد متصل هستید!

نکات مهم:
  • برای قطع اتصال، دوباره روی دکمه کلیک کنید
  • برای تست سرعت سرورها، Right Click → Test Server Speed
  • اپلیکیشن در System Tray (کنار ساعت) اجرا می‌شود
  • در صورت مشکل فایروال، آن را غیرفعال کنید

راهنمای اتصال در macOS

آموزش نصب و راه‌اندازی V2ray روی سیستم‌های مک

گام 1️⃣: نصب اپلیکیشن

🔹 اپلیکیشن پیشنهادی: Hiddify Next

📥 دانلود:

دانلود Hiddify-Next

(فایل Hiddify-MacOS.dmg را دانلود کنید)

یا گزینه‌های دیگر:

  • ClashX
  • Qv2ray

گام 2️⃣: نصب

  1. فایل .dmg را باز کنید
  2. اپلیکیشن را به پوشه Applications بکشید
  3. برای اولین بار: System Preferences → Security → "Open Anyway"
  4. اپلیکیشن را باز کنید

گام 3️⃣: افزودن اشتراک

در Hiddify Next:

  1. روی "➕ New Profile" کلیک کنید
  2. لینک اشتراک را کپی کنید
  3. گزینه "Add from Clipboard" را انتخاب کنید
  4. روی "Add" کلیک کنید

در ClashX:

  1. روی آیکون ClashX در Menu Bar کلیک کنید
  2. "Config" → "Remote Config" → "Manage"
  3. روی "Add" کلیک کنید
  4. لینک اشتراک را وارد کنید و روی OK کلیک کنید

گام 4️⃣: اتصال

  1. روی آیکون اپلیکیشن در Menu Bar کلیک کنید
  2. یکی از سرورها را انتخاب کنید
  3. گزینه "Set as System Proxy" را فعال کنید
  4. آیکون رنگش تغییر می‌کند (نشان اتصال)

✅ تمام! حالا به اینترنت آزاد متصل هستید!

نکات مهم:
  • برای بروزرسانی سرورها: Update Remote Config
  • برای تست سرعت: Benchmark (در ClashX)
  • اپلیکیشن در Menu Bar (بالای صفحه) قرار دارد
  • از آخرین نسخه macOS استفاده کنید

راهنمای اتصال در Linux

آموزش نصب و راه‌اندازی V2ray روی سیستم‌های لینوکس

گام 1️⃣: نصب اپلیکیشن

🔹 اپلیکیشن پیشنهادی: Hiddify Next

📥 دانلود:

دانلود Hiddify-Next

(فایل Hiddify-Linux-x64.AppImage را دانلود کنید)

یا گزینه‌های دیگر:

  • Qv2ray
  • v2rayA (Web-based)

گام 2️⃣: نصب و اجرا

روش AppImage:

chmod +x Hiddify-Linux-x64.AppImage
./Hiddify-Linux-x64.AppImage

یا برای Debian/Ubuntu:

اگر فایل .deb دانلود کردید:

sudo dpkg -i hiddify-*.deb
sudo apt-get install -f

گام 3️⃣: افزودن اشتراک

در Hiddify Next:

  1. اپلیکیشن را باز کنید
  2. روی "➕ New Profile" کلیک کنید
  3. لینک اشتراک را کپی کنید
  4. گزینه "Add from Clipboard" را انتخاب کنید
  5. روی "Add" کلیک کنید

در Qv2ray:

  1. Group → Subscription Settings
  2. Add Subscription
  3. لینک را وارد کنید
  4. Update Subscription

گام 4️⃣: اتصال

  1. یکی از سرورها را از لیست انتخاب کنید
  2. روی دکمه Connect کلیک کنید
  3. حالت System Proxy را فعال کنید

✅ تمام! حالا به اینترنت آزاد متصل هستید!

نکات مهم:
  • برای اجرای خودکار: اپلیکیشن را به Startup اضافه کنید
  • در صورت مشکل با DNS: تنظیمات DNS را چک کنید
  • برای CLI: می‌توانید از v2ray-core مستقیماً استفاده کنید
  • از توزیع‌های LTS برای پایداری بیشتر استفاده کنید

سوالات متداول

پاسخ سوالات رایج درباره خرید فیلترشکن حجمی V2ray

V2ray یک پروتکل پیشرفته برای عبور از فیلترینگ است که امنیت و سرعت بالایی دارد. این پروتکل از رمزنگاری قوی استفاده می‌کند و روی تمام دستگاه‌ها از جمله آیفون، اندروید، ویندوز و مک قابل استفاده است. مزایای اصلی آن شامل سرعت بالا، پایداری اتصال، امنیت بیشتر و مقاومت در برابر تشخیص است.
بله! تمام پلن‌های آردیسک وی‌پی‌ان با کاربر نامحدود هستند. شما می‌توانید اشتراک خود را روی تعداد نامحدودی دستگاه (موبایل، تبلت، لپ‌تاپ، کامپیوتر) به صورت همزمان استفاده کنید. فقط توجه داشته باشید که حجم ترافیک بین همه دستگاه‌ها مشترک است.
برای خرید فیلترشکن آیفون کافیست وارد ربات تلگرام ما شوید، پلن مورد نظر را انتخاب کنید و پس از پرداخت، لینک اتصال را دریافت کنید. راهنمای نصب کامل نیز ارائه می‌شود. همچنین می‌توانید از بخش "راهنمای اتصال" در این صفحه، آموزش کامل نصب روی iOS را مشاهده کنید.
شما می‌توانید از طریق کارت به کارت، درگاه بانکی و ارز دیجیتال پرداخت خود را انجام دهید. تمام فرآیند خرید از طریق ربات تلگرام انجام می‌شود. پس از انتخاب پلن، ربات گزینه‌های پرداخت را به شما نمایش می‌دهد و می‌توانید روش مناسب خود را انتخاب کنید.
می‌توانید از طریق ربات تلگرام حجم اضافه تهیه کنید یا اشتراک جدید بخرید. همچنین با خرید پلن‌های سه ماهه می‌توانید از تخفیف ویژه بهره‌مند شوید. ربات به شما اطلاع می‌دهد که حجم شما در حال اتمام است و می‌توانید قبل از اتمام کامل، حجم جدید خریداری کنید.
